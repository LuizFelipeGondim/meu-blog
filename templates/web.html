HTML


INTRODUÇÃO


O curso de HTML abordará os principais e mais básicos recursos e estruturas para o desenvolvimento de uma plataforma web.

A linguagem HTML (Hypertext Markup Language) é uma linguagem de marcação que tem o objetivo de exibir, estruturar e organizar documentos web através de marcações especiais denominadas tags, para que possam ser exibidos de forma apropriada por meio dos browsers quando um usuário solicita uma página. Sendo assim, o entendimento dessa linguagem é fundamental para compreensão nossos próprios sites.
