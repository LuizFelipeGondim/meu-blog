FORMULÁRIOS


As tags de formulário são implementadas para obter informações do usuário, realizar cadastros, receber opiniões, realizar login, entre outros. Inicialmente, colocamos a tag form no documento e depois incluimos o conteúdo do formulário. Quando queremos criar campos de preenchimento, devemos utilizar a tag input, enquanto para o campo referente à mensagem, devemos utilizar a tag textarea.

A tag input possui o atributo type, que varia entre diversos tipos, veja alguns tipos abaixo. Também há o atributo placeholder, que é uma mensagem que ficará disponível na caixa do input enquanto nada for digitado.



  • input type="text" : Define um campo para receber qualquer caracter.

  • input type="email" : Define um campo para receber emails.

  • input type="password" : Define um campo para receber senhas.

  • input type=”submit” : Define um botão que servirá para o envio do formulário.

  • input type="date" : Define um campo para recerber datas.

    • As tags textarea têm a mesma função que o input. No entanto, seu campo de preenchimento é um área extensa que pode ser redimensionada pelo usuário.
