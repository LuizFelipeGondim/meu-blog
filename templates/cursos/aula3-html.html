MULTIMEDIA


As tags de multimídia servem para incluir imagens, vídeos, áudios, iframes e outros tipos de conteúdo multimídia. Elas são:

  • video : Essa tag insere um vídeo no nosso documento. Para manipulá-la é necessário algumas ferramentas, dentre elas estão o atributo constrols e a tag source, servindo para ativar os controles de multimedia e rebecer o local do video, respectivamente.


  • audio : Essa tag funciona da mesma forma que a tag video, portanto, utilizamos a tag source dentro dela. Vejamos o exemplo a seguir:


  • img : Utilizada para apresentar imagens. Elas são acompanhadas pelos atributos src e alt. O primeiro serve para indicar o local da imagem e o segundo para definir um texto alternativo para imagem.


  • Por fim, os iframes que servem para incluir recursos de uma outra página. Geralmente adicionamos videos do youtube por meio deles.
