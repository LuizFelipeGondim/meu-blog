PRINCIPAIS FUNÇÕES


Funções são um conjunto de códigos que visam um resultado. Dessa forma, há algumas funções nativas no Visualg que nos auxiliará na construção do nossos códigos. Elas são utilizadas, geralmente, para diminuir o tamanho do código e torná-lo mais fluído. Segue as mais utilizadas:

  • Escreva(" ") : Mostra o texto que está escrito entre suas aspas na tela do computador.

  • Escreval(" ") : Faz a mesma coisa que a função escreva, a única diferença é que ela pula uma linha.

  • Leia( VARIÁVEL ) : Lê o que o usuário digitou e armazena na variável que está entre seus parênteses.

  • <- : Esse símbolo não é propriamente uma função, mas nos auxíliará bastante. Ele encaminha um valor para uma variável.



    • VARIÁVEIS


    Agora que conhecemos as funções mais utilizadas no Portugol, precisamos conhecer as variáveis, para começar-mos de vez a programar.

    As variáveis são um recurso que nossos programas utilizam para armazenar dados na memória do computador para que sejam manipulados quando conveniente.

    Toda variável deve começar por uma letra e depois conter letras, números ou underline, até um limite de 30 caracteres. Posteriormente, são seguidas por dois pontos e seu tipo de dado. Elas são declaradas da seguinte forma:


    Existem quatro tipos de dados primitivos, algumas linguagens subdividem estes tipos de dados em outros de acordo com a capacidade de memória necessária para cada variável, mas no geral, os tipos de dados primitivos são:


  • INTEIRO : Este é o tipo de dados para valores numéricos negativos ou positivos, sem casas decimais.


  • REAL : Este é o tipo de dados para valores numéricos negativos ou positivos, com casas decimais.


  • LÓGICO : Este tipo de dados pode assumir apenas dois valores VERDADEIRO ou FALSO.


  • CARACETER: Tipo de dados para variáveis que armazenam caracteres.


    • Outro tipo de variável bastante utilizada são os vetores. Um vetor ou variável bidimensional, é normalmente usadas para armazenar várias variáveis do mesmo tipo, aumenta a eficiencia dos nossos programas. Por exemplo, para armazenar o nome de 50 alunos, podemos criar apenas um vetor de tamanho 50 no lugar de cinquenta variáveis "aluno".
