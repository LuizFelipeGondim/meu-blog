ESTRUTURA PRINCIPAL HTML


Todo documento HTML apresenta elementos entre os sinais de "<" e ">". Esses elementos são as tags HTML, que são os comandos de formatação da linguagem. A maioria das "etiquetas" tem sua correspondente de fechamento, representada com uma "/". Por exemplo:


Isso é necessário porque as etiquetas servem para definir a formatação de uma porção de texto, e assim marcando o inicio e o fim com a formatação especificada por elas. No entanto, há algumas tags que não possuem fechamento, conhecidas como Self closing tags. Um exemplo disso é o br, utilizado para quebrar linhas.



NOTA: Podemos utilizar qualquer tipo de editor de texto para construirmos nossas páginas HTML. Então fique a vontade para escolher um editor que mais te agrade. Lembrando que para executarmos nosso trabalho, o documento precisa ter a extensão .html, por exemplo: pagina.html.


Há algumas tags essenciais que formam a estrutura básica de uma página. No HTML5, o documento padrão recebe a seguinte estrutura:



  • DOCTYPE html5 : A tag !DOCTYPE informa ao browser a versão do HTML que está sendo utilizada no documento. Nesse caso, a versão é a 5.

  • html : Tag que contêm todos os elementos do seu documento. Sendo assim, todo documento HTML deve iniciar e finalizar com essa tag.

  • head : Essa tag delimita o cabeçalho do documento. Nelas são armazenados os metadados da página de maneira invisível pra o usuário, aparacendo apenas o título na aba do navegador.

  • title : Essa tag define o título da sua página, o nome que aparecerá na aba.

  • meta : Essa tag permite inserir metadados ao seu documento.

  • body : Finalmente, a tag que contêm a parte vísivel de uma página.

    1. O que é html? E qual as tags básicas para a estrutura principal?


      HTML é uma linguagem de marcação que tem o objetivo de exibir, estruturar e organizar documentos web através de marcações especiais denominadas tags, para que possam ser exibidos de forma apropriada por meio dos browsers quando um usuário solicita uma página. As tags básicas são: html, head, body e title.
