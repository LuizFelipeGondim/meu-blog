SISTEMA DE ARQUIVOS


Teoricamente, um sistema de arquivo é um conjunto de estruturas lógicas que permite ao sistema operacional ter acesso e controlar os dados gravados no disco. Cada sistema operacional lida com um sistema de arquivos diferente e cada sistema de arquivos possui as suas peculiaridades, como limitações, qualidade, velocidade, gerenciamento de espaço, entre outras características. É o sistema de arquivos que define como os bytes que compõem um arquivo serão armazenados no disco e de que forma o sistema operacional terá acesso aos dados. Suas principais funções são o gerenciamento de arquivos, a navegação pela estrutura de diretórios, o acesso a arquivos e pastas, a recuperação de dados e o armazenamento de dados.

Podemos classificar os sistemas de arquivos em uso atualmente em quatro grandes grupos:


  • Sistemas de arquivos de Disco : É um sistema que aproveita a capacidade da mídia de armazenamento em disco para endereçar dados aleatoriamente em um curto período de tempo.

  • Sistemas de arquivos de Redes : É um sistema de arquivos que atua como um cliente para um protocolo de acesso remoto a arquivos, fornecendo acesso a arquivos em um servidor.

  • Sistemas de arquivos de Bancos de Dados : Sistema onde os arquivos são identificados por suas características, como tipo de arquivo, tópico, autor ou metadados ricos semelhantes.

  • Sistemas de arquivos de propósito especial : São os demais sistemas de arquivos.
